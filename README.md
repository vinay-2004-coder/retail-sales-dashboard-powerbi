# 📊 Retail Sales Dashboard (Power BI)

## 🚀 Overview

This project is an interactive Power BI dashboard designed to analyze retail sales performance across transactions, profit, returns, and regional distribution.
It provides a clear view of business performance using KPIs, trend analysis, and geographic insights.

---

## 🎯 Key Features

* KPI tracking for **Transactions, Profit, and Returns**
* Custom KPI: **Average Revenue per Transaction**
* Product brand performance analysis using matrix visualization
* Geographic analysis with interactive map
* Revenue trend analysis over time
* Country-level performance using treemap
* Dedicated insights page summarizing key business findings

---

## 🧠 Skills Demonstrated

* Power BI dashboard design
* DAX calculations (CALCULATE, DATEADD, DIVIDE)
* Time intelligence analysis
* Data modeling and relationships
* Data visualization best practices
* Business insight generation

---

## 🛠️ Tools Used

* Power BI
* DAX (Data Analysis Expressions)
